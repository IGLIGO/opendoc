# class B - 系统操作`0x07`

## 0xFE.设备重启

| 操作 | 接口索引 | 操作数  | 参数   |
| ---- | ---- | ---- | ---- |
| 重启 | 0x07 | 0xFE | xxx |

> 例： 
> `APP->:^,route,fc,length=2,0x07,0xFE,_check_`

Note: 移动端可通过判断与设备之间蓝牙连接断开即为成功重启

## 0xFF.设备关机

| 操作 | 接口索引 | 操作数  | 参数   |
| ---- | ---- | ---- | ---- |
| 关机 | 0x07 | 0xFF | xxx |

> 例： 
> `APP->:^,route,fc,length=2,0x07,0xFF,_check_`

Note: 移动端可通过判断与设备之间蓝牙连接断开即为成功关机
